# Autism-Activity-Recognition
# Autism Activity Recognition using MMASD Dataset

This repository contains my implementation and experimental work on autism activity recognition using the MMASD (Multimodal Dataset for Autism Intervention Analysis) benchmark dataset.

The project explores multimodal deep learning approaches for recognizing autism-related intervention activities using motion and pose-based behavioral features. The implementation focuses on activity understanding from privacy-preserving representations such as 2D/3D skeletal data and temporal movement patterns.

## Project Overview

The goal of this work is to investigate how deep learning and computer vision techniques can assist in behavioral activity analysis within autism intervention settings.

The implementation includes:

* Data preprocessing and feature handling
* Model training and evaluation
* Activity classification experiments
* Multimodal behavioral analysis workflows
